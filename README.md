# NSY Team Skills

Claude skills shared across the NSY Digital team. Upload these into your Claude AI Projects as knowledge files.

## Skills

| Skill | Purpose |
|-------|---------|
| [nsy-creative-strategy](nsy-creative-strategy/SKILL.md) | Write ad scripts, hooks, concepts, statics, briefs |
| [nsy-creative-research](nsy-creative-research/SKILL.md) | Run deep creative research for clients |
| [nsy-brief-builder](nsy-brief-builder/SKILL.md) | Populate the Official NSY Brief Template in Notion |
| [nsy-onboarding](nsy-onboarding/SKILL.md) | Onboard a new NSY Digital client end-to-end |
| [nsy-proposal-builder](nsy-proposal-builder/SKILL.md) | Build client audit and creative strategy proposals |
| [nsy-static-ad-generator](nsy-static-ad-generator/SKILL.md) | Generate AI static ad prompts for Nano Banana / Higgsfield |

## How to use

1. Open your Claude AI Project
2. Go to Project Knowledge
3. Upload the `SKILL.md` file from the skill you want
4. If the skill has a `references/` folder, upload those files too
5. Start a conversation and the skill is active

## Structure

Each skill folder contains:
- `SKILL.md` - the main skill file
- `references/` - supporting reference docs (where applicable)
